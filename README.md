# Metallprofil Price Tracker

Локальный MVP для отслеживания прайсов `Металл Профиль`.

На текущем этапе автоматизирована только группа:

- `Сэндвич-панели`

## Как это работает

1. Вы кладете новые PDF в папку `input/`.
2. Скрипт берет последний загруженный PDF и предыдущий для сравнения.
3. Из PDF вытаскиваются:
   - прайс по сэндвич-панелям
   - фасонные изделия для ТСП
   - стандартные цвета
   - минимальный заказ
4. Генерируются:
   - `data/sandwich-panels.json`
   - `site/index.html`

## Запуск

```powershell
python scripts/update_sandwich_panels.py
```

## Структура

- `input/` — входные PDF
- `data/` — распарсенные данные
- `scripts/` — парсер и генерация
- `site/` — готовая страница для GitHub Pages
